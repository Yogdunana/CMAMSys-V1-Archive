## 🎯 核心自动化功能

### 1. 全自动化流程协调器

**文件**：`src/services/auto-process-coordinator.ts`

**功能描述**：
实现从赛题输入到最终论文生成的完整自动化流程，包括以下阶段：

#### 1.1 群聊讨论阶段（自动化）
- 自动选择 2-3 个最优 AI Provider 参与讨论
- 最多 3 轮讨论，自动达成共识
- 自动生成任务列表
- 提取核心算法、创新点和可行性分析

#### 1.2 代码生成阶段（自动化）
- 基于讨论结果自动生成完整代码
- 支持 Python 和 MATLAB
- 自动包含数据预处理、核心算法、结果输出、可视化模块
- 自动添加注释和错误处理

#### 1.3 自动校验阶段（自动化）
- 语法校验：检查代码语法错误
- 结果校验：验证代码执行结果
- 可视化校验：检查图表生成
- 自动记录校验结果

#### 1.4 智能回溯机制（自动化）
- 最多 3 轮自动回溯
- 回溯策略：
  - 结果不理想 → 换思路（重新讨论）
  - 代码有问题 → 重新生成代码
  - 代码和结果都通过 → 进入论文生成
- 自动触发重新讨论和代码生成

#### 1.5 论文生成阶段（自动化）
- 基于讨论结果、代码和执行结果生成完整论文
- 支持多种竞赛格式（MCM、ICM、CUMCM 等）
- 支持中英文双语
- 自动生成摘要、引言、方法、结果、结论等章节

**自动化程度**：⭐⭐⭐⭐⭐ (95/100)
- 用户仅需输入赛题信息，系统自动完成所有步骤
- 无需人工干预，全程自动执行

### 2. AI Provider 自动选择

**文件**：`src/services/auto-provider-selector.ts`

**功能描述**：
基于赛题类型和任务类型自动选择最优 AI Provider

#### 2.1 智能标签系统

**能力标签**：
- 数学建模算法（mathematical_modeling）
- 代码生成（code_generation）
- 数据预处理（data_preprocessing）
- 优化算法（optimization_algorithm）
- 统计分析（statistical_analysis）
- 可视化设计（visualization）
- 批判性思维（critical_thinking）
- 问题求解（problem_solving）

**场景标签**：
- 美赛优化题（mcm_optimization）
- 美赛评价题（mcm_evaluation）
- 美赛预测题（mcm_prediction）
- 国赛优化题（cumcm_optimization）
- 国赛预测题（cumcm_prediction）
- 数据分析（data_analysis）
- 可视化设计（visualization_design）
- 数值模拟（numerical_simulation）

#### 2.2 自动选择算法

```typescript
1. 根据竞赛类型匹配场景标签
2. 根据问题类型匹配场景标签
3. 根据任务类型匹配能力标签
4. 计算每个 Provider 的匹配分数
5. 选择分数最高的 Provider
```

**自动化程度**：⭐⭐⭐⭐⭐ (90/100)
- 完全自动选择，无需人工配置
- 智能匹配算法，确保选择最优 Provider
