## 🔴 严重安全问题（5个）

### 1. JWT Secret 硬编码默认值
- **文件**：`src/lib/jwt.ts`
- **问题**：JWT Secret 有硬编码的默认值 `'your-super-secret-jwt-key-change-in-production'`
- **影响**：攻击者可以使用默认值生成有效的 JWT token，完全绕过身份验证
- **优先级**：🔴 紧急（立即修复）

### 2. CSRF Secret 硬编码默认值
- **文件**：`src/lib/csrf.ts`
- **问题**：CSRF Secret 有硬编码的默认值 `'your-super-secret-csrf-key-change-in-production'`
- **影响**：攻击者可以使用默认值生成有效的 CSRF token，绕过 CSRF 保护
- **优先级**：🔴 紧急（立即修复）

### 3. Token 可通过 URL 查询参数传递
- **文件**：`src/lib/auth-middleware.ts`
- **问题**：Token 可以通过 URL 查询参数提取
- **影响**：Token 会被记录在日志中、保存在浏览器历史记录中、被分享
- **优先级**：🔴 紧急（立即修复）

### 4. 加密密钥回退到 JWT Secret
- **文件**：`src/lib/encryption.ts`
- **问题**：加密密钥回退到使用 JWT Secret
- **影响**：如果 JWT Secret 使用默认值，加密完全失效，API Keys 等敏感数据可以被解密
- **优先级**：🔴 紧急（立即修复）

### 5. Content-Security-Policy 过于宽松
- **文件**：`src/lib/api-middleware.ts`
- **问题**：CSP 策略允许 `unsafe-inline` 和 `unsafe-eval`
- **影响**：允许内联脚本和动态代码执行，XSS 攻击更容易成功
- **优先级**：🔴 高危（立即修复）

---

## 🟠 高优先级问题（3个）

### 6. CSRF 保护默认关闭
- **文件**：`src/lib/api-middleware.ts`
- **问题**：很多预设的 CSRF 保护默认关闭
- **影响**：大部分 API 端点缺少 CSRF 保护，容易受到 CSRF 攻击
- **优先级**：🟠 高危（尽快修复）

### 7. 错误消息可能泄露用户存在性
- **文件**：`src/app/api/auth/login/route.ts`
- **问题**：错误消息可能泄露用户是否存在
- **影响**：攻击者可以通过暴力枚举确定哪些邮箱已注册
- **优先级**：🟠 高危（尽快修复）

### 8. 用户列表 API 缺少分页和限制
- **文件**：`src/app/api/admin/users/route.ts`
- **问题**：GET /api/admin/users 没有分页和限制
- **影响**：可能导致 DoS 攻击或数据泄露
- **优先级**：🟠 高危（尽快修复）

## 🟡 中优先级问题（3个）

### 9. PBKDF2 迭代次数可提升
- **文件**：`src/lib/encryption.ts`
- **问题**：PBKDF2 迭代次数为 100,000
- **影响**：对于现代硬件来说可能不够
- **优先级**：🟡 中等（建议修复）

### 10. 开发环境输出敏感日志
- **文件**：`src/lib/jwt.ts`
- **问题**：开发环境输出敏感信息
- **影响**：可能泄露敏感信息
- **优先级**：🟡 中等（建议修复）

### 11. 环境变量示例文件包含敏感信息
- **文件**：`.env.example`
- **问题**：包含敏感信息的示例值
- **影响**：用户可能直接使用示例值
- **优先级**：🟡 中等（建议修复）

---

## 🟢 低优先级问题（2个）

### 12. bcrypt 轮次可提升
- **文件**：`src/lib/password.ts`
- **问题**：bcrypt 轮次为 12
- **影响**：可以增加到 13-14 以增强安全性
- **优先级**：🟢 低危（可选修复）

### 13. Next.js allowedDevOrigins 配置
- **文件**：`next.config.ts`
- **问题**：allowedDevOrigins 允许 `*.dev.coze.site`
- **影响**：在生产环境中可能过于宽松
- **优先级**：🟢 低危（可选修复）
