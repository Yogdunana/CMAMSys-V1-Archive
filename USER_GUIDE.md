## 账户管理

### 个人资料

1. 点击右上角头像 → **个人资料**
2. 可以修改：
   - 用户名
   - 邮箱
   - 头像
   - 个人简介

### 修改密码

1. 点击右上角头像 → **账户设置**
2. 点击 **修改密码**
3. 输入当前密码和新密码
4. 点击 **保存**

### 启用双因素认证（2FA）

为提高账户安全性，建议启用 2FA：

1. 点击右上角头像 → **账户设置**
2. 找到 **双因素认证**
3. 点击 **启用**
4. 扫描 QR 码或输入密钥
5. 验证一次性密码
6. 保存备用代码（重要！）

## 团队协作

### 创建团队

1. 导航到 **团队** 页面
2. 点击 **创建团队**
3. 填写团队信息：
   - 团队名称
   - 团队描述
   - 团队类型（公开/私有）
4. 点击 **创建**

### 邀请成员

1. 进入团队详情页
2. 点击 **邀请成员**
3. 输入成员邮箱
4. 选择成员角色：
   - **所有者**：完全控制权限
   - **管理员**：管理成员和任务
   - **成员**：参与任务和讨论
   - **观察者**：只读权限
5. 点击 **发送邀请**

### 角色权限说明

| 功能 | 所有者 | 管理员 | 成员 | 观察者 |
|------|--------|--------|------|--------|
| 创建/删除团队 | ✅ | ❌ | ❌ | ❌ |
| 管理成员 | ✅ | ✅ | ❌ | ❌ |
| 创建任务 | ✅ | ✅ | ✅ | ❌ |
| 编辑论文 | ✅ | ✅ | ✅ | ❌ |
| 查看任务 | ✅ | ✅ | ✅ | ✅ |
| 导出论文 | ✅ | ✅ | ✅ | ❌ |

## 建模任务

### 创建新任务

1. 导航到 **建模任务** 页面
2. 点击 **创建任务**
3. 填写任务信息：
   - **任务名称**
   - **任务描述**
   - **竞赛类型**（MCM/ICM/CUMCM）
   - **截止日期**
   - **团队成员**
4. 选择 **AI Provider**（可选）
5. 点击 **创建**

### 任务流程

CMAMSys 自动化建模流程包含以下阶段：

```
1. 数据预处理
   ↓
2. 智能讨论
   ↓
3. 代码生成
   ↓
4. 代码验证
   ↓
5. 论文生成
   ↓
6. 论文优化
```

### 查看任务进度

1. 进入任务详情页
2. 查看 **进度面板**：
   - 当前阶段
   - 完成百分比
   - 剩余时间
   - AI 消耗统计

### 参与讨论

1. 在任务详情页点击 **讨论** 标签
2. 查看智能讨论内容
3. 可以：
   - 查看所有讨论轮次
   - 追踪任务列表
   - 查看生成代码
   - 查看验证结果

## AI Provider 配置

### 支持的 AI Provider

CMAMSys 支持以下 AI 服务：

| Provider | 说明 | 推荐场景 |
|----------|------|---------|
| **阿里云百炼** | 阿里云大模型服务 | 中文任务 |
| **火山引擎（豆包）** | 字节跳动大模型 | 通用场景 |
| **DeepSeek** | 深度求索大模型 | 数学和推理 |
| **OpenAI** | GPT-4 等模型 | 高精度需求 |

### 添加 AI Provider

1. 导航到 **设置** → **AI Provider**
2. 点击 **添加 Provider**
3. 填写配置：
   - **Provider 类型**
   - **API Key**
   - **端点 URL**（如果需要）
   - **模型名称**
   - **优先级**（数字越小优先级越高）
4. 点击 **保存**

### API Key 安全

- ✅ API Key 会使用 AES-256-GCM 加密存储
- ✅ 只有管理员可以查看和编辑
- ✅ 支持密钥轮换
- ⚠️ 请勿与第三方分享 API Key

## 论文管理

### 查看论文

1. 导航到 **论文** 页面
2. 选择要查看的论文
3. 可以：
   - 在线编辑
   - 查看版本历史
   - 对比版本差异
   - 恢复历史版本

### 导出论文

支持的导出格式：

| 格式 | 说明 |
|------|------|
| **Word (.docx)** | Microsoft Word 文档 |
| **PDF** | 便携式文档格式（中文支持） |

导出步骤：

1. 进入论文详情页
2. 点击 **导出**
3. 选择导出格式
4. 等待生成完成
5. 下载文件

### 版本管理

- **自动保存**：每次编辑都会自动保存版本
- **版本对比**：可以对比任意两个版本的差异
- **恢复版本**：可以恢复到任意历史版本
- **版本标签**：可以为重要版本添加标签

## 常见问题

### Q: 忘记密码怎么办？

A: 使用登录页面的"忘记密码"功能，系统会发送重置邮件到您的邮箱。

### Q: 如何提高建模任务的准确性？

A: 建议操作：
1. 提供清晰详细的任务描述
2. 配置多个 AI Provider 并设置优先级
3. 启用回溯机制，让 AI 自我纠正
4. 参与讨论过程，提供反馈

### Q: 任务失败了怎么办？

A: 任务失败的可能原因：
- API Key 配置错误
- 网络连接问题
- 数据格式错误
- AI 服务限流

解决方法：
1. 查看 **错误日志**
2. 检查 AI Provider 配置
3. 检查网络连接
4. 重新启动任务

### Q: 如何联系技术支持？

A: 您可以通过以下方式联系我们：
- 📧 Email: support@cmamsys.com
- 📝 Issue: https://github.com/Yogdunana/CMAMSys/issues
- 💬 在线客服: 工作时间 9:00-18:00

### Q: 是否支持离线使用？

A: 部分功能支持离线：
- ✅ 论文编辑
- ✅ 版本管理
- ❌ AI 功能（需要网络）
- ❌ 团队协作（需要网络）

## 最佳实践

### 团队协作建议

1. **明确分工**：根据成员专长分配角色
2. **定期同步**：使用讨论功能保持沟通
3. **版本控制**：重要节点添加版本标签
4. **备份论文**：定期导出备份

### 建模任务建议

1. **详细描述**：提供清晰的题目要求
2. **数据准备**：确保数据格式正确
3. **多 Provider**：配置多个 AI Provider 增强可靠性
4. **逐步验证**：关注每个阶段的输出

### 安全建议

1. **启用 2FA**：保护账户安全
2. **定期更新密码**：每 3-6 个月更新一次
3. **保护 API Key**：不要在公共代码仓库中存储
4. **及时更新**：保持系统版本最新
