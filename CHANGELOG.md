## 版本说明

### 版本命名规则

- **主版本号（Major）**：重大功能变更或不兼容的 API 变更
- **次版本号（Minor）**：向下兼容的功能性新增
- **修订号（Patch）**：向下兼容的问题修正

### 文档版本

文档版本与对应代码版本保持一致：

| 文档版本 | 代码版本 | 状态 |
|---------|---------|------|
| v2.1.0 | v2.1.0 | ✅ 当前版本 |
| v2.0.0 | v2.0.0 | ✅ 稳定版本 |
| v1.2.0 | v1.2.0 | ✅ 历史版本 |
| v1.1.0 | v1.1.0 | ✅ 历史版本 |
| v1.0.0 | v1.0.0 | ✅ 历史版本 |

### 文档类型说明

- ✅ **新增（Added）**：新功能、新文档、新配置
- 🔄 **改进（Changed）**：现有功能的改进或优化
- 🗑️ **移除（Removed）**：已移除的功能或配置
- ⚠️ **弃用（Deprecated）**：即将移除的功能
- 🐛 **修复（Fixed）**：Bug 修复
- 🔒 **安全（Security）**：安全相关的修复
